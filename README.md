Example usage:

```
echo "10,10,20,20" | bboxconvert -i xyxy -i normalized-center -w 640 -h 320
0.0234375,0.046875,0.015625,0.03125
```

```
echo "10,10,20,20" | bboxconvert -i xyxy -i yolo -w 640 -h 320
0.0234375,0.046875,0.015625,0.03125
```

```
printf "10,10,20,20\n10,10,20,30\n" | cargo run -- -i bb -o yolo -w 640 -h 320
0.0234375,0.046875,0.015625,0.03125
0.0234375,0.0625,0.015625,0.0625
```
